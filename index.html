Análisis de Datos de Plataforma de E-commerce

Este proyecto analiza el comportamiento de los usuarios en una plataforma de comercio electrónico. Evalúa el flujo de navegación, las órdenes de compra y los costos por fuente de adquisición para generar métricas clave y fundamentar decisiones estratégicas.

Se implementan procesos de limpieza de datos (fechas, columnas, duplicados), enriquecimiento temporal (día, semana, mes), y cálculo de métricas de negocio como CAC (Costo de Adquisición de Cliente), tasa de conversión e ingresos mensuales.

Tecnologías: Python (Pandas, Matplotlib, Seaborn, NumPy, SciPy), Jupyter Notebook.

Resultados Clave: El informe muestra resultados positivos generales, con un LTV mayor que el CAC, indicando ganancias por encima del margen del 50%. Las fuentes de marketing con mayor uso generan menor retorno, siendo la fuente 3 la de mayor costo y menor retorno, mientras que la fuente 10 tiene el menor costo y mayor retorno. Se destaca la necesidad de volumen de clientes, ya que la alta tasa de conversión del primer día (72.18%) sugiere que la prioridad debe ser atraer usuarios a la página. El ingreso promedio por venta es de $8.62, y el valor de vida promedio del cliente es de $18.92, lo que implica que realizan más de una compra.

Gym User Churn Prediction with Machine Learning

Este proyecto utiliza técnicas de Machine Learning para predecir si un usuario de gimnasio cancelará su servicio. A través de análisis exploratorio de datos (EDA), ingeniería de características y entrenamiento de modelos de clasificación, el objetivo es proporcionar a la empresa una herramienta para anticipar la baja de clientes y facilitar la toma de decisiones proactiva para mejorar la retención.

Tecnologías: Python (Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn), Jupyter Notebook, Análisis Estadístico.

Resultados Clave: El modelo Random Forest alcanzó una precisión del 91%, con un recall del 81.3% y una precisión del 83.8%. El AUC-ROC fue de 0.97, indicando una excelente discriminación y una fuerte capacidad para identificar usuarios propensos a la baja, lo que permite estrategias de fidelización más efectivas. Se identificaron factores clave que predicen la retención de clientes y el modelo puede implementarse para señalar posibles cancelaciones futuras.

Análisis de Datos de Telecomunicaciones

Este proyecto se enfoca en el análisis exhaustivo de datos de telecomunicaciones para extraer insights valiosos y fundamentar la toma de decisiones estratégicas. A través de la manipulación y el análisis de los datos, buscamos optimizar operaciones, identificar áreas de mejora y potenciar el rendimiento general de los servicios de telecomunicaciones.

Tecnologías: Python (Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn), Jupyter Notebook, Análisis Estadístico.

Resultados Clave: Se logró un preprocesamiento efectivo de los datos, asegurando la calidad del análisis. La función identificar_operadores_a_deshabilitar permite una evaluación objetiva del rendimiento y tiene la capacidad de identificar operadores ineficaces, siendo adaptable a diferentes umbrales de ineficacia. Los gráficos facilitan la interpretación de los resultados y se genera un dataset limpio para uso posterior.

A/A Test for Food Products

Este proyecto explora el comportamiento del consumidor en un experimento de prueba A/A para productos alimenticios. El objetivo principal es validar la consistencia de las variantes experimentales antes de pasar a una prueba A/B, asegurando que las diferencias observadas no se deban al azar.

Se realizó un análisis exhaustivo para validar la aleatorización del experimento y evaluar las diferencias estadísticas en el comportamiento entre grupos idénticos, estableciendo una base sólida para futuras pruebas A/B.

Tecnologías: Python (Pandas, NumPy, Seaborn, Matplotlib, Scipy Stats), Análisis Estadístico.

Resultados Clave: Se observaron tasas de conversión promedio en rangos como 0.45, 0.5 y 0.7 en los eventos analizados. Crucialmente, no se encontraron diferencias significativas entre los grupos, lo cual es exactamente lo que se espera de una prueba A/A exitosa, validando la aleatorización del experimento.
